# Overview
My analysis of artificial blood pressure and other data as part of Geoff's UCSF training

## Short Description
Generate and view a histogram for blood pressure. 

## Long Description
This repository contains workbooks and data to anlyze blood pressure readings for a patient population. The data provided here is randomly generated and is not drawn from any actual patient population. You should be able to convert this for analysis of read data.
